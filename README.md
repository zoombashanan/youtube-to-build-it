# YouTube to BUILD-IT
A web app that turns any YouTube tutorial into a step-by-step instructional guide.

## Status
Pre-build. Initialization complete. Features pending.

## Stack
- Next.js (App Router)
- Vercel (hosting)
- Supabase (auth + database)
- Resend (magic link emails)
- Anthropic Claude Sonnet 4.6 (BUILD-IT generation)
